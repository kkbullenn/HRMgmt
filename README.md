# HRMgmt

HRMgmt is an ASP.NET MVC web application designed to streamline human resource operations such as employee management, payroll handling, and shift oversight. It provides administrators and HR staff with a centralized system for managing workforce data efficiently.

---

## 🚀 Overview

This system helps organizations manage core HR processes in one place. It supports employee records, scheduling, payroll tracking, and administrative tools for monitoring staff operations.

---

## 🧰 Tech Stack

* **Framework:** ASP.NET MVC
* **Language:** C#
* **IDE:** Visual Studio
* **Database:** (Configured locally through project settings)

---

## 📦 Getting Started

### Prerequisites

* Visual Studio (recommended)
* .NET SDK compatible with the project

### Installation & Run

1. Clone the repository:

   ```bash
   git clone https://github.com/kkbullenn/HRMgmt.git
   ```
2. Open the solution file in **Visual Studio**
3. Build the solution
4. Run the project using **Start Debugging (F5)** or **Start Without Debugging (Ctrl + F5)**

No additional setup or configuration is required.
